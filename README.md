# Lista de Tarefas

Aplicação web desenvolvida com HTML, CSS e JavaScript.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Adicionar tarefas
- Marcar como concluída
- Remover tarefas
- Salvar automaticamente no navegador (LocalStorage)
