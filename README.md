# 📋 Lista de Tarefas

Aplicação web desenvolvida com HTML, CSS e JavaScript para gerenciamento de tarefas.

## 🚀 Demonstração

🔗 Projeto online: (https://lista-tarefas-blue-one.vercel.app)

## 💻 Tecnologias

- HTML5
- CSS3
- JavaScript
- LocalStorage

## ✨ Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Marcar como concluídas
- Excluir tarefas
- Persistência dos dados com LocalStorage
